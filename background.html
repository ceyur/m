Свойства фона

background-color: цвет; – задаёт цвет фона элемента
background-color: mediumaquamarine;

background-image: градиент■или■url('адрес картинки'); – устанавливает фоновое изображение или градиент для элемента
Создать градиент – https://www.colorzilla.com/gradient-editor/
background-image: url('https://apple.com');

background-position: координаты; – задаёт начальное положение фонового изображения (background-image)
background-position: 25px 30px;

background-size: значение■или■размеры; – масштабирует фоновое изображение согласно заданным размерам
Значения:
1. auto – если задано одновременно для ширины и высоты (auto auto), размеры фона остаются исходными; если только для одной стороны картинки (100px auto), то размер вычисляется автоматически исходя из пропорций картинки
2. cover – масштабирует изображение с сохранением пропорций так, чтобы его ширина или высота равнялась ширине или высоте блока
3. contain – масштабирует изображение с сохранением пропорций таким образом, чтобы картинка целиком поместилась внутрь блока
background-size: contain;

background-repeat: значение; – задаёт повторение фонового изображения (background-image)
Значения:
1. no-repeat – устанавливает одно фоновое изображение без повторений, положение которого определяется свойством background-position (по умолчанию в левом верхнем углу)
2. repeat – фоновое изображение повторяется по горизонтали и вертикали
3. repeat-x – фоновый рисунок повторяется только по горизонтали
4. repeat-y – фоновый рисунок повторяется только по вертикали
5. inherit – наследует значение родителя
6. space – изображение повторяется столько раз, чтобы полностью заполнить область; если это не удаётся, между картинками добавляется пустое пространство
7. round – изображение повторяется столько раз, чтобы в области поместилось целое число рисунков; если это не удаётся сделать, то фоновые рисунки масштабируются
background-repeat: repeat;

background-attachment: значение; – устанавливает, будет ли прокручиваться фоновое изображение вместе с содержимым элемента
Значения:
1. fixed – делает фоновое изображение элемента неподвижным
2. scroll – позволяет перемещаться фону вместе с содержимым
3. inherit – наследует значение родителя
4. local – фон фиксируется с учётом поведения элемента. Если элемент имеет прокрутку, то фон будет прокручиваться вместе с содержимым, но фон выходящий за рамки элемента остаётся на месте
background-attachment: fixed;

background-clip: значение; – устанавливает, где будет отображаться фон
Значения:
1. border-box - фон отображается до внешнего края границы элемента
2. padding-box - фон отображается до внутреннего края границы элемента
3. content-box - фон отображается только внутри контента
background-clip: border-box;

background: background-color background-image background-position/background-size background-repeat background-attachment; – сокращённая запись для свойств background-color, background-image, background-position, background-size, background-repeat, background-attachment
background: #9cdaff url('img/bg.jpg') center/50% 50% fixed, aquamarine url('img/bg1.jpg') left/25% 25% no-repeat;
