свойство: значение; – объяснение свойства, его особенности Значения: 1. 2. пример (значение по умолчанию) Подобные свойства: 1. 2.

Обозначения: 1. ↓ – свойство наследуется 2. ■ – объединение

Значения, приминяющиеся для всех свойств: 1. inherit – элемент должен унаследовать родитесльское значение 2. initial – элемент должен использовать значение по умолчанию (не браузерное) 3. revert – элемент должен использовать браузерное значение, а не по умолчанию 4. revert-layer – откатывает значение свойства к значению свойства в предыдущем каскадном слое 5. unset – для наследуемых свойств значение работает так же, как inherit; для ненаследуемых свойств значение работает так же, как initial

Псевдоклассы

элемент:hover {свойства элемента} – свойства в фигурных скобках начинают работать при наведении на элемент div:hover { color: mediumaquamarine; }

элемент:active {свойства элемента} – срабатывает при нажатии на элемент .button:active { padding: 25px; }

элемент:visited {свойства элемента} – срабатывает только для посещённых элементов a:visited { text-decoration: underline; }

элемент:focus {свойства элемента} – срабатывает для элементов, находящихся в фокусе (форм, полей ввода) form:focus { border: 2px solid #539; }

элемент:first-child {свойства элемента} – срабатывает для первого элемента в блоке li:first-child { margin: 25px; }

элемент:last-child {свойства элемента} – срабатывает для последнего элемента в блоке ol:last-child { padding: 5px; }

элемент:nth-child(порядковый номер) {свойства элемента} – срабатывает для элемента в блоке, чей порядковый номер указан в псевдоклассе div:nth-child(4) { width: 50px; }

элемент:odd {свойства элементов} – срабатывает для элементов с нечётными номерами .end:odd { background-color: #000; }

элемент:even {свойства элементов} – срабатывает для элементов с чётными номерами #background:even { border: 0; }
